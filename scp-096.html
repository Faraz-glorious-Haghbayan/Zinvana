SCP-096 — The Shy Guy

Containment class: Euclid
Core vibePowerhouse / Last resort / Shock response
Main edgeDecisive follow-through
Main riskHard to de-escalate

Quick read

Euclid

You’re quiet until something crosses the line—then you go all-in. When activated, you act with overwhelming intensity.

Use this page like a mirror: keep the strengths, tame the watch-outs.

Strengths + watch-outs

Strengths (how you win)

  • Decisive follow-through
  • Raw drive
  • Protects what matters

Watch-outs (how you slip)

  • Hard to de-escalate
  • Overreaction risk
Growth line: Build a pause before action—control turns intensity into precision.

Drills (use today)

These are simple on purpose. Simple is repeatable.

  • Notice the trigger
  • Take one breath
  • Choose the smallest effective response

Where you shine

  • No turning back
  • Finishing what started

Roles you naturally fall into

  • Powerhouse
  • Last resort
  • Shock response

FAQ

  • Is this “canon” SCP lore? This is a personality-style profile inspired by SCP entries — built for fast learning, not a full SCP article.
  • How should I use the result? Pick one drill, run it for 3 days, then re-check which watch-out shows up most.
  • Why containment class here? It’s a metaphor for predictability: Safe = stable, Euclid = uncertain, Keter = high-risk.
