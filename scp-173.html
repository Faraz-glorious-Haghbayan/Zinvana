SCP-173 — The Sculpture

Containment class: Euclid
Core vibeProtector / Rule-setter / First responder
Main edgeHigh vigilance
Main riskCan become rigid

Quick read

Euclid

You thrive on attention to detail and decisive boundaries. Under pressure you become hyper-focused: observe, lock down, and prevent the next mistake.

Use this page like a mirror: keep the strengths, tame the watch-outs.

Strengths + watch-outs

Strengths (how you win)

  • High vigilance
  • Clear boundaries
  • Fast containment decisions

Watch-outs (how you slip)

  • Can become rigid
  • May over-rely on control
Growth line: Keep the standards, but communicate the why—people follow rules they understand.

Drills (use today)

These are simple on purpose. Simple is repeatable.

  • Name the single biggest risk
  • Set one simple rule
  • Confirm the team understands

Where you shine

  • Securing the perimeter
  • Stopping chaos early

Roles you naturally fall into

  • Protector
  • Rule-setter
  • First responder

FAQ

  • Is this “canon” SCP lore? This is a personality-style profile inspired by SCP entries — built for fast learning, not a full SCP article.
  • How should I use the result? Pick one drill, run it for 3 days, then re-check which watch-out shows up most.
  • Why containment class here? It’s a metaphor for predictability: Safe = stable, Euclid = uncertain, Keter = high-risk.
