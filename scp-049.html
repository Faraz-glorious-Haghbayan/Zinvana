SCP-049 — The Plague Doctor

Containment class: Euclid
Core vibeSpecialist / Diagnosis lead / Process enforcer
Main edgeStrong mission focus
Main riskCan become fixated

Quick read

Euclid

You see problems everywhere—and you feel responsible to fix them. You’re methodical, intense, and convinced there’s a correct way forward.

Use this page like a mirror: keep the strengths, tame the watch-outs.

Strengths + watch-outs

Strengths (how you win)

  • Strong mission focus
  • Calm under pressure
  • Persistence

Watch-outs (how you slip)

  • Can become fixated
  • May ignore dissenting signals
Growth line: Keep your conviction, but test assumptions—certainty improves when it meets evidence.

Drills (use today)

These are simple on purpose. Simple is repeatable.

  • Write down 2 alternative explanations
  • Ask one skeptical question
  • Re-check the outcome

Where you shine

  • Sticking to the mission
  • Refining the procedure

Roles you naturally fall into

  • Specialist
  • Diagnosis lead
  • Process enforcer

FAQ

  • Is this “canon” SCP lore? This is a personality-style profile inspired by SCP entries — built for fast learning, not a full SCP article.
  • How should I use the result? Pick one drill, run it for 3 days, then re-check which watch-out shows up most.
  • Why containment class here? It’s a metaphor for predictability: Safe = stable, Euclid = uncertain, Keter = high-risk.
